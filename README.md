# Hotelier Database Handoff
1. Download latest release of Hotelier
2. Download database.sqlite

## On Windows Computer

1. Install Hotelier.
2. Open Hotelier once, then close it.
3. Press:

```text
Win + R
```

4. Paste this path:

```text
%APPDATA%\Hotelier\database
```

5. Press Enter.
6. Replace the existing file:

```text
database.sqlite
```

with the `database.sqlite` file you downloaded here.

7. Open Hotelier again.

The app should now use the passed database.
